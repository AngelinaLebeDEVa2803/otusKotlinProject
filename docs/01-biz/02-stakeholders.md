### Стейкхолдеры:

| Стейкхолдер | Категория | Уровень Влияния | Уровень Интереса | Ключевые Потребности/Опасения | Метрики Влияния на Проект |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Рядовые сотрудники (Пользователи)** | Пользователь | Средний | **Критически Высокий** | Простота бронирования, прозрачность (видеть коллег), стабильность приложения, защита приватности. | User Satisfaction (NPS/CSAT), Daily Active Users (DAU), Retention Rate. |
| **Офис-менеджеры/Администраторы** | Пользователь | Высокий | **Критически Высокий** | Удобство управления планом офиса, аналитика для оптимизации пространства, инструменты для решения споров. | Adoption Rate среди админов, время на управление офисом, % использования мест (Utilization Rate). |
| **Руководители отделов** | Пользователь | Высокий | Высокий | Инструменты для координации команды (бронь для группы), видимость присутствия команды, интеграция с календарем. | % команд, использующих групповое бронирование, Satisfaction Score. |
| **HR-департамент (Internal)** | Команда (Внутр.) | Высокий | Высокий | Данные для формирования гибкой политики работы (hybrid work), улучшение employee experience, привлекательность офиса для сотрудников. | Employee Satisfaction scores, данные об использовании офиса. |
| **IT-департамент (Internal)** | Команда (Внутр.) | **Критический** | Высокий | Безопасность данных, интеграция с корпоративными системами (Active Directory, календари), стабильность, масштабируемость, низкие затраты на поддержку. | Uptime (99.9+%), время отклика API, количество инцидентов, успешность интеграций. |
| **Финансовый департамент (Internal)** | Команда (Внутр.) | Высокий | Средний | Снижение затрат на аренду офисных площадей (благодаря оптимизации), обоснование ROI от внедрения приложения, контроль бюджета на разработку/подписку. | Utilization Rate офиса, расчет экономии на кв. метрах, CAC (если приложение продается). |
| **Инвесторы/Высшее руководство** | Финансы/Управление | **Критический** | Высокий | Рост эффективности использования ресурсов (оптимизация площадей), повышение продуктивности сотрудников, окупаемость инвестиций (ROI). | Utilization Rate, Cost Saving на аренде, Employee Productivity metrics. |
| **Владелец продукта/Product Manager** | Управление | **Критический** | **Критический** | Удовлетворенность пользователей, достижение бизнес-целей (рост метрик), рентабельность продукта, реализация стратегии. | Все ключевые продуктовые метрики (DAU, Retention, Satisfaction), OKR/KPI. |
| **Разработчики (Internal/External)** | Команда (Внутр./Внеш.) | Высокий | Средний | Четкие требования, современный tech stack, возможность профессионального роста, стабильные процессы (Agile/Scrum). | Velocity, Time to Market, Code Quality, Bug Rate. |
| **Юристы (Internal)** | Команда (Внутр.) | Высокий | Средний | Соответствие ФЗ-152 "О персональных данных" (хранение данных о перемещениях сотрудников), юридическая чистота пользовательских соглашений. | Отсутствие предписаний и штрафов от Роскомнадзора. |
| **Фитнес- и кейтеринг-партнеры в офисе** | Внешний (Партнер) | Низкий | Средний | Прогнозируемость нагрузки (чтобы готовить нужное количество кофе/обедов) на основе данных о бронированиях. | Качество интеграции (если есть), доступ к данным о посещаемости. |
| **Регуляторы (Роскомнадзор)** | Внешний | **Критический** | Низкий | Соблюдение законодательства РФ о защите персональных данных (ФЗ-152). | Наличие предписаний, штрафов. |
